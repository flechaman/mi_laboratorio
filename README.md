# ejercicios_github

Laboratorio personal para practicar Git, scripts, infraestructura y proyectos.

## Estructura

- `infra/`: configuraciones de infraestructura.
- `projects/`: proyectos o ejercicios separados.
- `scripts/`: utilidades y automatizaciones.
- `docs/`: documentacion del laboratorio.
- `workspace/`: area de trabajo temporal.
