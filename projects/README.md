# Projects

Proyectos completos o prototipos que pueden crecer de forma independiente.

## Estructura sugerida por proyecto

```text
project-name/
  README.md
  src/
  tests/
  .env.example
  requirements.txt
```

## Proyectos iniciales

- `python-automation-starter/`: base para automatizaciones locales.
- `azure-docker-lab/`: pruebas de Azure, contenedores y despliegue.
- `ai-local-rag/`: laboratorio para RAG local.
- `grafana-observability/`: dashboards, metricas y alertas.
- `powerbi-datasets/`: datasets preparados para PowerBI.
