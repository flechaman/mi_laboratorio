# Scripts

Scripts reutilizables, automatizaciones y pruebas historicas.

## Carpetas

- `python/learning/`: scripts existentes de practica y aprendizaje.
- `automation/`: automatizaciones locales o de sistema.
- `azure/`: utilidades relacionadas con Azure.
- `gmail/`: scripts para Gmail API.

Los scripts deben poder ejecutarse desde Windows 11 con PowerShell y Python instalado.
