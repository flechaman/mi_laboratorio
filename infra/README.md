# Infra

Configuracion reutilizable de infraestructura, contenedores y servicios auxiliares.

## Carpetas

- `docker/`: Dockerfiles, compose parciales y configuracion comun.
- `azure/`: plantillas, scripts y notas para recursos de Azure.
- `grafana/`: provisioning, dashboards y Prometheus.
- `rag/`: configuracion de vector stores, modelos locales y pipelines RAG.
- `powerbi/`: configuracion y notas de integracion con PowerBI.
- `gmail-api/`: ubicacion esperada para credenciales locales de Gmail API.

No guardar credenciales reales en esta carpeta.
